# 📂 Data Folder

This folder is reserved for the Open University Learning Analytics Dataset (OULAD).

⚠️ Raw data files are not included in this repository due to size and licensing restrictions.

To reproduce the analysis locally:
1. Download the dataset from: https://analyse.kmi.open.ac.uk/open_dataset
2. Place the CSV files inside `data/raw/`.

If the raw data are missing, the notebook will raise a clear error explaining how to obtain them.
